# cn-portal-notes

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为通用 HTML 页面的存储与展示空间。

项目旨在提供一个简洁、稳定的页面托管环境，方便维护和分享静态内容。

## 目录结构

```
/
├── pages/           # 存放所有独立 HTML 页面
├── assets/          # 公共资源文件（CSS、图片、字体等）
├── README.md        # 本文件
└── index.html       # 可选：项目入口页面
```

- `pages/`：每个子文件夹或文件代表一个独立的 HTML 页面，命名清晰便于识别。
- `assets/`：公共资源，可被多个页面引用，减少重复文件。

## 页面归档说明

- 所有 HTML 页面均为独立文件，不依赖后端服务。
- 页面内容以静态信息展示为主，不含动态交互功能。
- 归档内容可能来自不同来源，已做基础整理，但保持原始结构与样式。
- 部分页面可能包含外部资源链接（如 CDN 文件），建议在离线环境下自行检查可用性。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会记录在提交历史中。
- 如需提交页面，请遵循以下基本规则：
  - 页面文件使用 `.html` 后缀，编码为 UTF-8。
  - 避免在页面中使用外部不可控资源（如第三方跟踪脚本）。
  - 页面文件名应简洁，反映内容主题。
- 欢迎通过 Issue 提出建议或报告问题，但请注意本仓库不提供技术支持。

## 许可

本仓库内容仅供学习与参考，不构成任何形式的承诺或保证。具体页面可能包含各自的版权声明，请以页面内信息为准。
